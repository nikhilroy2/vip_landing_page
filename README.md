# Aura Enterprise - VIP SME Landing Page

A premium, highly professional landing page designed for an exclusive boutique advisory firm targeting the ambitious SME (Small and Medium Enterprise) sector.

## Features

- **Premium Aesthetics:** Dark mode styling with elegant gold/bronze accents and glassmorphism paneling.
- **Sophisticated Typography:** Uses *Playfair Display* for headings and *Montserrat* for body text to ensure maximum readability with a premium feel.
- **Modern Ambient Graphics:** Subtle noise overlays, floating golden orbs, and architectural grid backgrounds create depth and a high-end visual experience.
- **Interactive UI:**
  - Dynamic Glassmorphism Navigation Bar that transitions on scroll.
  - Smooth scrolling for internal anchor links.
  - Custom Intersection Observer animations (fade-in up, revealing cards).
  - Number counting animation for the statistics section.
- **Eye-Catching "Aura Advantage" Section:** Luminous glowing cards that highlight key strategic differentiators.
- **High-Quality Contextual Imagery:** Integrated with highly professional executive stock imagery from Unsplash.
- **Clean Structure & Semantic HTML:** Built with standard HTML5, pure CSS3 (no external heavy frameworks), and Vanilla JS for high performance. 

## File Structure

- `index.html`: The main semantic HTML document containing the structure and German copy.
- `styles.css`: The extensive bespoke stylesheet managing the layout, grid systems, custom properties (CSS variables), animations, and responsive breakpoints.
- `script.js`: Lightweight JavaScript handling smooth scroll, navbar scroll state, number ticker animations, and scroll reveals. 

## Setup

No build tools are required. Simply serve the directory using any local web server to preview:

```bash
# Using python 3
python -m http.server 8125

# Then open http://localhost:8125 in your browser
```
